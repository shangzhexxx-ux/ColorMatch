# Color Match Live — Product

## Product Purpose

A free, browser-based tool for creating Xiaohongshu (Little Red Book) style typographic photo cards. Users upload any photo, the app extracts dominant colors, applies editorial typography overlays, and exports a polished social-media-ready image.

## Users

- **Primary**: Chinese social media creators (Xiaohongshu, WeChat Moments)
- **Secondary**: General content creators wanting quick editorial-style cards
- **Tertiary**: Casual users wanting to make personal photos more shareable
- Users expect: fast, no-account-required, one-click polish, mobile-first experience

## Tone & Character

Warm, editorial, slightly cinematic. Not cold SaaS. The UI stays out of the way; the output is the star. Think: a well-designed photo printing service, not a complex design tool.

## Anti-references

- Not Canva (too complex, too many features)
- Not generic "AI photo enhancer" apps (cheap, garish)
- Not professional design tools (Figma, Photoshop)
- No dark mode by default — warm, light, inviting

## Core User Flow

1. User lands on page → sees upload prompt immediately
2. Upload photo → app processes (EXIF parsing, color extraction)
3. Preview appears with auto-generated color scheme + editable text overlay
4. User customizes: scheme presets, custom colors, font, location, date
5. Export → full-resolution image download

## Strategic Principles

- **Mobile-first**: Most users come from social media links, on phones
- **Zero friction**: No login, no accounts, no friction before value
- **Instant feedback**: Every color/font choice reflects live in preview
- **One-tap polish**: Auto-extract from EXIF so the output "just looks good"
- **Fast export**: Downloadable image should match preview exactly

## Register

Product — design serves the productivity/creation flow. Not a marketing page.
